Start Server with WebPack Watch?

WebPack can compile your REACTjs files automatically everytime you SAVE them. This can save a lot of time during the development of your app. If you would like WebPack to watch your files then start your server like this:

  1. Open a seperate terminal
  2. Navigate to your app directory (folder)
  3. Type the following lines of code:
    • npm run start                                ←
Your Reactjs files will now be automatically compiled by REACTjs each time you SAVE a file.
