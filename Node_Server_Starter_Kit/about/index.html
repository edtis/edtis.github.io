Production boilerplate to start developing react components (with WebPack).

Bonus: Use ES6 or regular javascript.

Ready for small to medium production app.


Starter Kit Comes with

  1. EJS
  2. 404 error handler
  3. 500 error handler
  4. Routes directory
  5. UncaughtException error handler [Bonus]

Boilerplate is

  1. Ready for Socket.io
  2. Ready for MongoDB
  3. Ready for ES6

EJS

EJS will help you transfer dynamic data in to your html code from your node server [Server to client].

404 Error handler

If a user tries to access a page that is not on the server your 404 error handler will catch it and you can serve up a webpage to display. The 404 error handler is simply a function and in that function you can do things (whatever you like) such as (but not limited too) server a 404 page.

If you need a design template for a 404 page then check out some of our (pre-made) designs for 404 pages here.
